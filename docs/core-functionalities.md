# Core Functionalities

This list keeps you focused on the required product scope without drifting into unnecessary features.

## 1. Account creation and account management

What it should do:

- allow a developer to register
- allow login and logout
- allow the developer to view and update their own profile
- protect authenticated parts of the app

Minimum fields:

- display name
- email
- short bio
- skills or tech stack

## 2. Create a project entry

What it should do:

- allow a logged-in developer to create a project post
- capture what they are building
- capture the current project stage
- capture what support they need

Suggested fields:

- title
- description
- project stage
- support required
- tags or technologies
- created date
- owner id

## 3. Live feed of developer activity

What it should do:

- show recently created projects
- show recent project progress updates
- allow users to discover what others are building

The feed should answer:

- who is building
- what they are building
- what stage they are in
- whether they need help

## 4. Comments on projects or updates

What it should do:

- allow authenticated users to comment on feed items or projects
- show discussion in context
- prevent empty or invalid comments

## 5. Collaboration request / raise hand

What it should do:

- allow a developer to express interest in helping on a project
- store who requested to collaborate
- optionally include a short message
- allow the project owner to review requests later

## 6. Project progress updates

What it should do:

- allow the project owner to post milestone updates over time
- show progress in chronological order
- keep the feed alive with continuous activity

Suggested examples:

- MVP completed
- authentication finished
- deployed first version

## 7. Project completion

What it should do:

- allow a project owner to mark a project as completed
- change the project status from active to completed
- trigger or create a celebration record

## 8. Celebration Wall

What it should do:

- display completed projects
- highlight developers who finished building in public
- act as a positive closing state for the user journey

## Nice-to-have only if time remains

- search or filter by technology
- likes or reactions
- profile avatars
- timestamps formatted nicely
- dark styling polish
